Detailed Description

embARC Example for secure echoclient based on FreeRTOS+LwIP and MatrixSSL

Extra Required Tools

Extra Required Peripherals

  • Digilent PMOD WIFI(MRF24WG0MA)

Design Concept

This example is designed as a secure echoclient. The communication is based on TLS. This example needs to work with echoserver example of matrixssl in pair.

Usage Manual

  • Hardware:
    • EMSK node1: PMOD WiFi to J5 on EMSK
    • EMSK node2: PMOD WiFi to J5 on EMSK
  • WiFi Network: SSID:embARC, PASSWORD:qazwsxedc
  • Run steps:
    • This example should run together with the EMBARC_APP_MATRIXSSL_FREERTOS_ECHOSERVER.
    • echoserver should run first, get the IP from the uart output message such as "ipaddr 192.168.43.252".
    • change the IP of echoserver in echoclient source code, build and run this example(echoclient).
    • the exchange of information will be shown in terminal as below.
