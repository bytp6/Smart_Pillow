  • Digilent PMOD RF2(MRF24J40)

Design Concept

This node works as a border router between 802.15.4 based IPv6 network and Wi-Fi based IPv4 network. In IPv6 network, the data packets whose destination is not in IPv6 network are forwarded to IPv4 network. Meanwhile, one entry is set up in the NAT table of router, like: IPv6 address: IPv6 port : IPv4 address ：IPv4 port : map port (start from 1024). The data packets from IPv4 netwrok are forwarder to IPv6 netwokr according to the NAT table.
